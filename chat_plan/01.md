[Task 1] 채팅 관련 코드 구조 파악

지금 코드베이스에서 다음을 찾아서 정리해줘:

1. 채팅 페이지/레이아웃
   - app/ 또는 pages/ 밑에서 채팅 화면을 담당하는 파일 목록
     (예: app/(chat)/rooms/[roomId]/page.tsx 등)
   - 각 파일에서 채팅 UI 루트 컴포넌트 이름 (예: <ChatRoomPage>, <ChatLayout>)

2. 메시지 리스트 관련 컴포넌트/훅
   - 현재 "메시지를 반복 렌더링"하는 부분이 어디에 있는지 찾고,
     해당 컴포넌트/훅 이름과 경로를 나열해줘.
   - 아직 없다면, "어디에 새로 만들기 좋은지" 후보 경로를 제안해줘.
     (예: src/features/chat/components/ChatMessageList.tsx)

3. 데이터 로딩/Realtime 파이프라인
   - Supabase 클라이언트 초기화 코드 위치 (예: src/entities/supabase/client.ts)
   - 현재 채팅 메시지를 가져오는 쿼리 함수/훅(useQuery/useInfiniteQuery 등) 위치
   - Supabase Realtime(Postgres Changes) 구독을 어디서 하는지
     (이미 있으면 관련 코드 경로/함수명, 없으면 "없다"고 명시)

4. TanStack Virtual 사용 위치
   - 게시글/상점에서 @tanstack/react-virtual을 사용하는 코드들을 찾아서
     - 파일 경로
     - 어느 컴포넌트에서 어떤 방식으로 virtualizer를 쓰는지
     를 요약해줘.
   - 여기서 가져올 수 있는 "공통 추상화 패턴"이 있으면 같이 제안해줘
     (예: scroll container 컴포넌트, 스크롤 바 스타일, etc).

출력:
- "채팅 관련 엔트리 포인트 목록"
- "현재 메시지 리스트 렌더링 위치 및 방식"
- "Supabase Realtime/Query 파이프라인 요약"
- "TanStack Virtual 사용 패턴 요약"
을 각각 소제목으로 나눠서 정리해라.
